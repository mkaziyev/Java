1) http://localhost:8080/index.jsp - начальная страница,
При нажатии на кнопке Show Users происходит переход на страницу ользователей
2) http://localhost:8080/users_list - страница пользователей
На этой странице отображаются все пользователи,
можно их добавлять, удалять, редактировать.
При добавлении и удалении пользователя запись добавляется(удаляется) в таблицу и страница перезагружается с новыми данными.
При редактировании пользователя происходит переход на страницу http://localhost:8080/edit/{id}.
При нажатии на кнопку Search users происходит переход на страницу showUsers
3) http://localhost:8080/edit/{id} - страница редактирования
На этой странице можно редактировать даныые пользователя,
При нажатии на кнопке Edit, данные сохраняются, и открывается страница Edited page.
С этой страны можно перейти обратно ны страницу пользователей.
4) http://localhost:8080/searchUsers - страница поиска.
